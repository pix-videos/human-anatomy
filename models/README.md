# 3D Models for Human Anatomy Project

Place your GLB model files in this directory. The following models are needed:

## Required Models

1. **heart.glb** - Human heart model
2. **brain.glb** - Human brain model
3. **lungs.glb** - Human lungs model

## Recommended Sources

### Free Models (CC0/CC-BY Licensed)

1. **Sketchfab** - High-quality anatomical models
   - Heart: https://sketchfab.com/3d-models/realistic-human-heart-3f8072336ce94d18b3d0d055a1ece089
   - Brain: https://sketchfab.com/search?q=human+brain&type=models&features=downloadable
   - Lungs: https://sketchfab.com/3d-models/realistic-human-lungs-ce09f4099a68467880f46e61eb9a3531

2. **NIH 3D Models** - Medical/scientific models
   - https://3d.nih.gov/discover
   - Search for "heart", "brain", "lungs"
   - Many models available in GLB format

3. **Human Reference Atlas** - Anatomically accurate models
   - https://humanatlas.io/3d-reference-library
   - Requires account/login for downloads
   - CC BY 4.0 licensed

## Download Instructions

1. Visit the recommended sources above
2. Filter by "Free" and "Downloadable"
3. Download GLB format (or convert from other formats)
4. Rename files to match the required names above
5. Place files in this `models/` directory

## File Format

- **Format**: GLB (preferred) or GLTF
- **Size**: Keep under 10MB per file for optimal loading
- **License**: Ensure models are CC0 or CC-BY licensed for free use
